## 사용한 기술 스택

- HTML
- CSS

<br/>

### 구현 기능

- Wordle 게임 Rule 구현 (자릿수와 알파벳이 맞으면 초록색, 자릿수만 맞으면 노란색, 모두 아니면 회색)
- 키보드 or 마우스 입력 처리
- 실제 존재하는 단어인지 체크

<br/>

### 구현하고 싶은 기능
 
- 결과 공유 기능 (스냅샷)
- 기록 관리
- 대전 모드
